Genetic Algorithm Simulation

Fish evolution sim using a genetic algorithm (adapted from PyWorld), inspired by my experience at Haystagg, where I used genetic algorithms to model top user populations.

  • Simulates a predator-prey environment where fish evolve over generations using a genetic algorithm (based on PyWorld)
  • Each fish has a simple 22-bit genome controlling size, speed, vision, and reaction chance
  • Used survival time as the fitness score, with sharks applying constant pressure each generation
  • Shows behaviors emerging over time (smaller fish, better awareness, smarter reactions) without hardcoding them
  • Built with Python + pygame, with a clean separation between simulation loop and genetic logic
